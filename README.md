# מונורפו משרד הביטחון

- client: אפליקציית React עם Redux Toolkit
- server-dotnet: שרת ASP.NET Core 8 Web API

## איך מריצים?

### client
cd client
npm install
npm run dev

### server-dotnet
cd server-dotnet
dotnet restore
dotnet run

### server-node
cd server-node
npm install
npm install mongoose
node index.js
